## INSTALLATION

<b>Step 1</b> :

  Add it in your root build.gradle at the end of repositories:

	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}

<b>Step 2</b> :

Add the dependency

	dependencies {
	     implementation 'com.github.Anilpatil40:CustomDialog:Tag'
	}
